296  // Computes the DENORMS for bl(baseline) and cn(character) normalization
297  // during feature extraction. The input denorm describes the current state
298  // of the blob, which is usually a baseline-normalized word.
299  // The Transforms setup are as follows:
300  // Baseline Normalized (bl) Output:
301  // We center the grapheme by aligning the x-coordinate of its centroid with
302  // x=128 and leaving the already-baseline-normalized y as-is.
303  //
304  // Character Normalized (cn) Output:
305  // We align the grapheme's centroid at the origin and scale it
306  // asymmetrically in x and y so that the 2nd moments are a standard value
307  // (51.2) ie the result is vaguely square.
308  // If classify_nonlinear_norm is true:
309  // A non-linear normalization is setup that attempts to evenly distribute
310  // edges across x and y.
311  //
312  // Some of the fields of fx_info are also setup:
313  // Length: Total length of outline.
314  // Rx: Rounded y second moment. (Reversed by convention.)
315  // Ry: rounded x second moment.
316  // Xmean: Rounded x center of mass of the blob.
317  // Ymean: Rounded y center of mass of the blob.
318  static void SetupBLCNDenorms(const TBLOB& blob, bool nonlinear_norm,
319  DENORM* bl_denorm, DENORM* cn_denorm,
320  INT_FX_RESULT_STRUCT* fx_info);
